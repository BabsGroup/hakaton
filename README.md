# Бекэнд для голосового гида

Принимает запросы от приложения, распознает/синтезирует голос, собирает информацию о местах из osm, общается с GigaChat 

Как запустить:
```shell
# Запускаем БД
docker-compose up -d
# Запускаем flask
. .venv/bin/activate
python app.py
```

Приложение запущено на http://82.97.241.218:5000
